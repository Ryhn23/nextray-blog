Here are some of the cool things (self-proclaimed haha) I've been building across infrastructure, AI, web, and hardware:

### 1. Nextray Cluster
A multi-architecture, heterogeneous cloud infrastructure hub. Managed via Tailscale and Cloudflare, it serves as the secure backbone for my containerized deployments, AI workloads, and self-hosted services.
Link: [cluster.nextray.org](https://cluster.nextray.org)

### 2. SolarisNet
An interactive AI platform that goes beyond standard chatbots. It features custom character logic, sentiment analysis, image generation, and custom voice cloning to create truly interactive resonator AI.
Link: [snet.shorekeeper.me](https://snet.shorekeeper.me)

### 3. StoryGraph
A simple local-first worldbuilding platform for writers with zero-telemetry. It utilizes Transformer.js for in-browser semantic search and integrates with Ollama to provide an agentic AI assistant, ensuring 100% data privacy. 
Link: [story.shorekeeper.me](https://story.shorekeeper.me)

### 4. Vera & Etra
Far beyond standard AI agents, this is a highly complex, dual-faceted orchestrator system with distinct autonomous roles:
* **Vera (AIOps):** My personal infrastructure operator. Vera manages my entire Nextray cluster and various backend services directly through conversational commands via Discord and WhatsApp.
* **Etra (Artificial Human):** An AI designed for deep, organic interaction. Moving away from standard generative chat, Etra supports real-time voice calls, delivers highly natural, human-like responses, and is deployed across multiple platforms.

### 5. GMaps Data Scraper
A lightweight automation tool designed to autonomously extract certain location and business data from Google Maps, organizing the output directly into structured CSV files for data science and analysis workflows.

### 6. Desk Companion
An IoT smart desktop assistant built from scratch using dual ESP microcontroller boards, bridging the gap between my software systems and physical hardware.
